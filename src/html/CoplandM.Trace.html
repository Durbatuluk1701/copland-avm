Shuffles

A trace is a list of events. shuffle merges two traces as is done by parallel execution. The order of events in the traces to be merged does not change, but all other interleavings are allowed.

Inductive shuffle: list Ev → list Ev → list Ev → Prop :=
| shuffle_nil_left: ∀ es, shuffle [] es es
| shuffle_nil_right: ∀ es, shuffle es [] es
| shuffle_left:
    ∀ e es0 es1 es2,
      shuffle es0 es1 es2 →
      shuffle (e :: es0) es1 (e :: es2)
| shuffle_right:
    ∀ e es0 es1 es2,
      shuffle es0 es1 es2 →
      shuffle es0 (e :: es1) (e :: es2).
Hint Constructors shuffle : core.

Lemma shuffle_length:
  ∀ es0 es1 es2,
    shuffle es0 es1 es2 →
    length es0 + length es1 = length es2.

Lemma shuffle_in_left:
  ∀ e es0 es1 es2,
    shuffle es0 es1 es2 →
    In e es0 →
    In e es2.

Lemma shuffle_in_right:
  ∀ e es0 es1 es2,
    shuffle es0 es1 es2 →
    In e es1 →
    In e es2.

Lemma shuffle_in:
  ∀ e es0 es1 es2,
    shuffle es0 es1 es2 →
    In e es2 ↔ In e es0 ∨ In e es1.

Lemma shuffle_in_skipn_left:
  ∀ i e es0 es1 es2,
    shuffle es0 es1 es2 →
    In e (skipn i es0) →
    In e (skipn i es2).

Lemma shuffle_in_skipn_right:
  ∀ i e es0 es1 es2,
    shuffle es0 es1 es2 →
    In e (skipn i es1) →
    In e (skipn i es2).

Lemma shuffle_earlier_left:
  ∀ es0 es1 es2 e0 e1,
    earlier es0 e0 e1 →
    shuffle es0 es1 es2 →
    earlier es2 e0 e1.

Lemma shuffle_earlier_right:
  ∀ es0 es1 es2 e0 e1,
    earlier es1 e0 e1 →
    shuffle es0 es1 es2 →
    earlier es2 e0 e1.

Lemma shuffle_nodup_append:
  ∀ tr0 tr1 tr2,
    NoDup tr0 → NoDup tr1 →
    disjoint_lists tr0 tr1 →
    shuffle tr0 tr1 tr2 →
    NoDup tr2.

Big-Step Semantics

The traces associated with an annotated term are defined inductively.

Inductive trace: AnnoTerm → Plc →
                 list Ev → Prop :=
| tasp: ∀ r x p,
    trace (aasp r x) p [(asp_event (fst r) x p)]
| tatt: ∀ r x p q tr1,
    trace x q tr1 →
    trace (aatt r q x) p
          ((req (fst r) (fst r) p q (unanno x) )
             :: tr1 ++
             [(rpy (pred (snd r)) (pred (snd r)) p q)])
| tlseq: ∀ r x y p tr0 tr1,
    trace x p tr0 →
    trace y p tr1 →
    trace (alseq r x y) p (tr0 ++ tr1)
| tbseq: ∀ r s x y p tr0 tr1,
    trace x p tr0 →
    trace y p tr1 →
    trace (abseq r s x y) p
          ((split (fst r) p)
             :: tr0 ++ tr1 ++
             [(join (pred (snd r)) p)])
| tbpar: ∀ r s x y p tr0 tr1 tr2,
    trace x p tr0 →
    trace y p tr1 →
    shuffle tr0 tr1 tr2 →
    trace (abpar r s x y) p
          ((split (fst r) p)
             :: tr2 ++
             [(join (pred (snd r)) p)]).
Hint Resolve tasp : core.

Lemma trace_length:
  ∀ t p tr,
    trace t p tr → esize t = length tr.

The events in a trace correspond to the events associated with an annotated term, a place, and some evidence.

Lemma trace_events:
  ∀ t p tr v,
    well_formed t →
    trace t p tr →
    In v tr ↔ events t p v.
      Require Import StructTactics.

Lemma trace_range:
  ∀ t p tr v,
    well_formed t →
    trace t p tr →
    In v tr →
    fst (range t) ≤ ev v < snd (range t).

Lemma trace_range_event:
  ∀ t p tr i,
    well_formed t →
    trace t p tr →
    fst (range t) ≤ i < snd (range t) →
    ∃ v, In v tr ∧ ev v = i.

Lemma trace_injective_events:
  ∀ t p tr v0 v1,
    well_formed t →
    trace t p tr →
    In v0 tr → In v1 tr →
    ev v0 = ev v1 →
    v0 = v1.

Lemma nodup_trace:
  ∀ t p tr,
    well_formed t →
    trace t p tr →
    NoDup tr.

Event Systems and Traces


Lemma evsys_tr_in:
  ∀ t p tr ev0,
    well_formed t →
    trace t p tr →
    ev_in ev0 (ev_sys t p) →
    In ev0 tr.

The traces associated with an annotated term are compatible with its event system.

Theorem trace_order:
  ∀ t p tr ev0 ev1,
    well_formed t →
    trace t p tr →
    prec (ev_sys t p) ev0 ev1 →
    earlier tr ev0 ev1.

Copland specific event systems.

Require Import Omega Preamble More_lists Term Event_system.

Construct an event system from an annotated term, place, and evidence.

Fixpoint ev_sys (t: AnnoTerm) p: EvSys Ev :=
  match t with
  | aasp (i, j) x ⇒ leaf (i, j) (asp_event i x p)
  | aatt (i, j) q x ⇒
    before (i, j)
      (leaf (i, S i) (req i p q (unanno x)))
      (before (S i, j)
              (ev_sys x q)
              (leaf (pred j, j) (rpy (pred j) p q)))
  | alseq r x y ⇒ before r (ev_sys x p)
                          (ev_sys y p)
  | abseq (i, j) s x y ⇒
    before (i, j)
           (leaf (i, S i)
                 (split i p))
           (before (S i, j)
                   (before (S i, (pred j))
                           (ev_sys x p)
                           (ev_sys y p))
                   (leaf ((pred j), j)
                   (join (pred j) p)))
  | abpar (i, j) s x y ⇒
    before (i, j)
           (leaf (i, S i)
                 (split i p))
           (before (S i, j)
                   (merge (S i, (pred j))
                          (ev_sys x p)
                          (ev_sys y p))
                   (leaf ((pred j), j)
                   (join (pred j) p)))
  end.

Lemma evsys_range:
  ∀ t p,
    es_range (ev_sys t p) = range t.

Lemma well_structured_evsys:
  ∀ t p,
    well_formed t →
    well_structured ev (ev_sys t p).

The events in the event system correspond to the events associated with a term, a place, and some evidence.

Lemma evsys_events:
  ∀ t p ev,
    well_formed t →
    ev_in ev (ev_sys t p) ↔ events t p ev.

Maximal events are unique.

Lemma supreme_unique:
  ∀ t p,
    well_formed t →
    ∃ ! v, supreme (ev_sys t p) v.

Lemma evsys_max_unique:
  ∀ t p,
    well_formed t →
    unique (supreme (ev_sys t p)) (max (ev_sys t p)).


lseq is associative relative to the event semantics

Lemma lseq_assoc:
  ∀ t1 t2 t3 i p,
    same_rel
      (ev_sys (snd (anno (lseq t1 (lseq t2 t3)) i)) p)
      (ev_sys (snd (anno (lseq (lseq t1 t2) t3) i)) p).
